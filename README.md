##  Каталог товаров с категориями и фильтрацией товаров по FULLTEXT 

###  Проект размещён на  
``http://localhost:8080/``

### Шаги для развёртывания:  
``cp .env.example .env``   
``composer i``  
``php artisan key:generate``  
``docker compose up -d``   
``docker compose exec app sh -c "php artisan migrate"``
